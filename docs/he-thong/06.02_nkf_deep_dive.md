# 🔬 CHƯƠNG 2: NKF Deep Dive — Hợp nhất thần kinh-động lực

## Lý thuyết "Ngọn sóng"

Lực đánh = chuỗi các ngọn sóng nối tiếp:

- **Sóng 1**: Bàn chân → đùi
- **Sóng 2**: Đùi → hông
- **Sóng 3**: Hông → vai
- **Sóng 4**: Vai → vút tay → đầu vợt

Nếu Sóng 2 chưa đạt đỉnh mà Sóng 3 đã bắt đầu → "ngọn sóng" gãy → năng lượng triệt tiêu.

## Công thức tổng hòa

$$\text{Lực tổng hợp} = \sum_{k=1}^{K} A_k \cdot e^{-\frac{(t-t_k)^2}{2\sigma_{sync}^2}}$$

| Biến            | Ý nghĩa                                        |
| --------------- | ---------------------------------------------- |
| $A_k$           | Biên độ lực (chân > hông > vai > tay > cổ tay) |
| $t_k$           | Thời điểm đạt đỉnh                             |
| $\sigma_{sync}$ | Chỉ số khớp nhau                               |

**Tiêu chuẩn Elite**: $\sigma_{sync} \geq 0.92$ → đỉnh sóng chồng khít → "đỉnh tháp" năng lượng.

## EMD (Electromechanical Delay)

$$\Delta t_{EMD} = t_{signal} + t_{Ca} + t_{co}$$

~30-100ms. Elite **không đợi bóng đến mới ra lệnh** — họ **Anticipation** trước $\Delta t$.

## Gồng vs Kích hoạt

| Gồng (Tension)     | Kích hoạt (Activation)    |
| ------------------ | ------------------------- |
| Co cứng toàn bộ cơ | Co đúng nhóm cơ đúng lúc  |
| Nhiễu thần kinh    | Tối đa lực, tối thiểu mỏi |
| Chậm, chấn thương  | Tốc độ, kiểm soát         |

## sEMG Activation Map (chuẩn NKF ≥ 0.92)

| T (ms)  | Cơ kích hoạt            |
| ------- | ----------------------- |
| 0-20    | Core (nền móng)         |
| 20-50   | Đùi (đẩy GRF)           |
| 50-80   | Obliques (xoay hông)    |
| 80-110  | Vai + ngực (mở tay)     |
| 110-120 | Cổ tay khóa (chạm bóng) |

## Neural Noise

Nếu sEMG xuất hiện xung điện hỗn loạn (căng thẳng, gồng) → $\sigma_{sync}$ giảm → "đánh hụt", "lệch điểm ngọt".

## 3 Bài tập NKF

### Drill 1: Breath-Motion Sync

- Hít vào khi chuẩn bị (nạp)
- Thở ra mạnh + dứt khoát đúng khoảnh khắc tiếp bóng

### Drill 2: Visual-Motor Overlay

- Xem video slow-motion
- Nhắm mắt + tự mô phỏng trong não (Visualization)
- Thực hiện thật

### Drill 3: Rhythm Training (Metronome)

- Nhịp 1: Đạp đất
- Nhịp 2: Xoay hông
- Nhịp 3: Mở vai
- Nhịp 4: Vút

## KPI

- $\sigma_{sync} \geq 0.92$
- $\Delta t \leq 0.12s$
