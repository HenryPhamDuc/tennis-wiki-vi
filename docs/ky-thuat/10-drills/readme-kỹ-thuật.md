## 🔟 Hướng dẫn Setup

### Bước 1: Tạo cấu trúc thư mục

Tạo các thư mục như trong cấu trúc đã liệt kê.

### Bước 2: Cài Plugin cần thiết trong Obsidian

1. **Dataview** (bắt buộc) — `obsidian://show-plugin?id=dataview`
2. **Templater** (tùy chọn) — `obsidian://show-plugin?id=templater-obsidian`
3. **Calendar** (tùy chọn) — `obsidian://show-plugin?id=calendar`
4. **Tag Wrangler** (tùy chọn) — `obsidian://show-plugin?id=tag-wrangler`

### Bước 3: Cấu hình Dataview

Bật chế độ Dataview trong **Settings → Dataview → Enable JavaScript Queries**.

### Bước 4: Sử dụng Templates

Trong **Settings → Templater**, trỏ template folder đến `99_Templates/`.

### Bước 5: Tạo Daily Note

Trong **Settings → Daily Notes**, thiết lập:

- Date format: `YYYY-MM-DD`
- New file location: `98_Daily_Notes/`
- Template: `99_Templates/_Template_Nhat_Ky_Tap.md`
