**Drills for Developing Touch:\
Part 3**

**Dave Hagler**

This month in Part 3 of this series I am adding 4 new, more advanced drills for developing touch. [Click Here](https://www.tennisplayer.net/members/ultimate_drillgames/dave_hagler/drills_for_developing_touch/part_01/) for Part 1 and [Click Here](https://www.tennisplayer.net/members/ultimate_drillgames/dave_hagler/drills_for_developing_touch/part_02/) for Part 2.

+---+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
|   | **Emperor\'s Volley**                                                                                                                                                                                                                                                                                                 |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | I learned this drill from USPTA and PTR Master Professional Ken DeHart 30 years ago. Bump the ball up then volley it to your partner who will bump it and then volley it back.                                                                                                                                        |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | Now catch the ball on your forehand, toss it with the racquet without the bump hit to your partner who catches, tosses and hits back to you.                                                                                                                                                                          |
|   |                                                                                                                                                                                                                                                                                                                       |
|   |                                                                                                                                             |
+===+=======================================================================================================================================================================================================================================================================================================================+
|   | **Mr. G**                                                                                                                                                                                                                                                                                                             |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | When I was in my early 20\'s, I spent 20 months in exile in a hellish place known as the DC Suburbs. I worked at a tennis club that no longer exists.                                                                                                                                                                 |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | There was a man at the club who everyone called \"Mr. G.\" At the time there was a pro tournament sponsored by the Washington Star -- a paper that no longer exists. When the pro players saw him at the tournament, they all said. \"Hi, Mr. G.\"                                                                    |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | Mr. G taught me this drill and I still use it. When I coached college, at the end of the year my players filled out surveys telling me what they liked and what they didn\'t. This drill was the most and least favorite several times\--so those who liked it really liked it and those who didn\'t, really didn\'t. |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | The Mr. G drill is a surefire indicator of which side is your touch side. It\'s a partner drill. Start at the T, and hit forehand to forehand or backhand to backhand.                                                                                                                                                |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | Increase the angle on every shot until you and your partner are outside of the netposts, hitting almost parallel to the net. Now switch your hitting side and work your way back around all the way to the opposite net post.                                                                                         |
|   |                                                                                                                                                                                                                                                                                                                       |
|   |                                                                                                                                                    |
+---+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
|   | **2 Box Dink \'Em**                                                                                                                                                                                                                                                                                                   |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | This drill emphasizes lateral movement. One player drops the ball off the top of the net. The service boxes are the entire court. Whoever wins the point drops the next ball. Play to 7 points.                                                                                                                       |
|   |                                                                                                                                                                                                                                                                                                                       |
|   |                                                                                                                                                |
+---+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
|   | **1 Box Dink \'Em**                                                                                                                                                                                                                                                                                                   |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | 1 box Dink \'Em can actually be more a forwards\--backwards game than 2 box Dink \'Em. Use 1 service box on each side\--deuce to ad so the game is not diagonal. Again drop the ball off the net cord.                                                                                                                |
|   |                                                                                                                                                                                                                                                                                                                       |
|   | You can volley in 1 box Dink \'Em, but you may not hit down. You also can\'t volley the drop or the shot after the drop.                                                                                                                                                                                              |
|   |                                                                                                                                                                                                                                                                                                                       |
|   |                                                                                                                                             |
|   |                                                                                                                                                                                                                                                                                                                       |

Stay Tuned, more touch drills to come!                                                                                                                                                                                                                                                           |
